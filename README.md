# Web Socket Client

Web Socket Application created with Vite and SocketIO

## Instructions

1. Install all dependencies.

    ```bash
    yarn install
    ```

2. Start the [backend application](https://github.com/RicardoMaldonado93/nestjs-teslo-shop).

3. Start the application.

    ```bash
    yarn dev
    ```
